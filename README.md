# ELM327 OBD-II Console Tool


###  Overview

Simple Windows C++ console application for reading vehicle diagnostics via an ELM327 adapter.

- No dependencies
- Uses Windows API directly
- Works over COM ports

---

###  Features

‚ÄĘ Auto-detect COM port  
‚ÄĘ Auto-init ELM327  
‚ÄĘ Read live vehicle data  
‚ÄĘ Show DTC error codes  
‚ÄĘ List supported PIDs  
‚ÄĘ Live dashboard mode  

---

###  Supported Data

‚ÄĘ RPM  
‚ÄĘ Speed  
‚ÄĘ Coolant Temperature  
‚ÄĘ Intake Air Temperature  
‚ÄĘ Engine Load  
‚ÄĘ Throttle Position  
‚ÄĘ MAF  
‚ÄĘ Voltage  
‚ÄĘ DTC Codes  

---

### Requirements

‚ÄĘ Windows
‚ÄĘ C++ compiler (if self compilation is needed)
‚ÄĘ ELM327 adapter (USB preferred)  
‚ÄĘ OBD-II compatible vehicle  
‚ÄĘ Ignition ON  

---

### Usage

1. Plug in ELM327  
2. Turn ignition ON  
3. Run program  
4. Wait for detection  
5. Select option from menu  

Press **Q** to exit dashboard  

---

### Notes

‚ÄĘ Not all vehicles support all PIDs  
‚ÄĘ Cheap adapters may behave inconsistently  
‚ÄĘ Bluetooth adapters can be unreliable
